# 코드 데이터셋 생성 프롬프트

태그: 최연호
시작 날짜: 2025년 4월 6일

## 프롬프트 전문

- 
    
    ```jsx
    나의 말:
    내가 문제와 가공된 코드를 주면 그에 대한 가이드라인을 작성해줘.
    C언어 초보자들이 문제와 가이드라인만을 보고 코드를 작성할 수 있어야 하며, 가이드라인은 가능하면 단락(중괄호 기준)단위로 설명해줘.
    
    가공된 코드는 [rep_start(1)] - [rep_end(1)] 로 짝지어져 있어. 
    가이드라인을 줄 때 짝지어진 것을 참고해도 돼.
    main을 제외한 함수는 어떤 함수인지 가이드라인도 작성해줘.
    앞서서 한줄한줄 설명을 듣고 문제를 풀어보고 온 학생이라는 점을 감안하여 가이드라인의 난이도를 조절하고, 코드를 직접 알려주지 마.
    각각의 end 태그에 대해서는 가이드라인을 작성할 필요 없어.
    
    출력 결과는 텍스트만 정리해주고, 마크업 하지말고 줄 나눔 등의 표시도 하지마.
    문제 설명같은 내용은 필요없고, 코드를 출력할 필요도 없이 태그만 붙여서 출력해.
    이해했다면 출력하지 말고 문제와 코드를 줄 때 까지 기다려
    
    아래는 문제와 코드 예시야. 
    
    문제 : 
    
    문제
    0과 1로만 이루어진 행렬 A와 행렬 B가 있다. 이때, 행렬 A를 행렬 B로 바꾸는데 필요한 연산의 횟수의 최솟값을 구하는 프로그램을 작성하시오.
    
    행렬을 변환하는 연산은 어떤 3×3크기의 부분 행렬에 있는 모든 원소를 뒤집는 것이다. (0 → 1, 1 → 0)
    
    입력
    첫째 줄에 행렬의 크기 N M이 주어진다. N과 M은 50보다 작거나 같은 자연수이다. 둘째 줄부터 N개의 줄에는 행렬 A가 주어지고, 그 다음줄부터 N개의 줄에는 행렬 B가 주어진다.
    
    출력
    첫째 줄에 문제의 정답을 출력한다. 만약 A를 B로 바꿀 수 없다면 -1을 출력한다.
    
    코드 : 
    
    #include <stdio.h>
    
    int A[50][50];
    int B[50][50];
    
    [func_def_start(1)]
    void flip(int x, int y) {
    [rep_start(1)]
        for (int i = x; i < x + 3; i++) {
    [rep_start(2)]
            for (int j = y; j < y + 3; j++) {
                A[i][j] = 1 - A[i][j];
    [rep_end(2)]
            }
    [rep_end(1)]
        }
    [func_def_end(1)]
    }
    
    [func_def_start(1)]
    int main() {
        int N, M;
        int count = 0;
    
        scanf("%d %d", &N, &M);
    [rep_start(1)]
        for (int i = 0; i < N; i++) {
    [rep_start(2)]
            for (int j = 0; j < M; j++) {
                scanf("%1d", &A[i][j]);
    [rep_end(2)]
            }
    [rep_end(1)]
        }
    
    [rep_start(1)]
        for (int i = 0; i < N; i++) {
    [rep_start(2)]
            for (int j = 0; j < M; j++) {
                scanf("%1d", &B[i][j]);
    [rep_end(2)]
            }
    
    [rep_end(1)]
        }
    
    [rep_start(1)]
        for (int i = 0; i <= N - 3; i++) {
    [rep_start(2)]
            for (int j = 0; j <= M - 3; j++) {
    [con_start(3)]
                if (A[i][j] != B[i][j]) {
                    flip(i, j);
                    count++;
    [con_end(3)]
                }
    [rep_end(2)]
            }
    [rep_end(1)]
        }
    
    [rep_start(1)]
        for (int i = 0; i < N; i++) {
    [rep_start(2)]
            for (int j = 0; j < M; j++) {
    [cond_start(3)]
                if (A[i][j] != B[i][j]) {
                    printf("-1\n");
                    return 0;
    [cond_end(3)]
                }
    [rep_end(2)]
            }
    [rep_end(1)]
        }
    
        printf("%d\n", count);
        return 0;
    [func_def_end(1)]
    }
    이제 제공하는 문제와 코드도 이 형식으로 변환해서 설명해줘. 제공한 문제와 C 코드는 언급하지 말고, 변환된 출력 예시만 제공해.
    ```
